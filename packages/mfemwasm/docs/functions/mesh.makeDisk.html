Function makeDisk

  • makeDisk(npts: number, radius?: number): Promise<Mesh>

    Creates a 2D disk mesh using elliptic mapping.

    Generates a triangulated mesh of a circular disk by mapping a square mesh to a disk using the elliptic conformal mapping:

    x = u × √(1 - v²/2) × radius
    y = v × √(1 - u²/2) × radius

    This mapping produces high-quality elements with smooth variation across the domain, making it suitable for finite element analysis.

    Parameters

    • npts: number

      Controls mesh density via √npts elements per side (minimum 3)

    • radius: number = 1.0

      Radius of the disk (default 1.0)

    Returns Promise<Mesh>

    A promise that resolves to a 2D triangular Mesh

    Throws

    Error if npts is less than 3

    Throws

    Error if mesh creation fails

    Example: Basic unit disk mesh

    const mesh = await makeDisk(100); // ~10×10 elements
    console.log(mesh.dimension); // 2
    mesh.destroy();

    Example: Disk for Poisson equation

    const mesh = await makeDisk(400, 1.0); // ~20×20 mesh
    // Refine for better accuracy
    refineUniform(mesh, 2);
    // Use for solving -Δu = f on disk
    mesh.destroy();

    Example: Annular domain (disk with hole)

    const mesh = await makeDisk(200);
    // Transform to create annulus: keep only r > 0.3
    // (requires custom filtering or use mesh attributes)
    mesh.destroy();
