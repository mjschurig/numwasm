Function projectFunction

  • projectFunction(gf: GridFunction, fn: (coords: number[]) => number): void

    Projects a JavaScript function onto a grid function using nodal interpolation.

    The function is evaluated at the physical coordinates of each degree of freedom to determine the DOF values. For standard Lagrange elements, this corresponds to interpolation at the nodal points.

    Note: This implementation uses the MFEM FunctionCoefficient for projection. However, due to the complexity of passing JavaScript callbacks to C++, this function currently provides a simplified implementation. For complex functions, consider computing DOF values directly and using gf.setData().

    Parameters

    • gf: GridFunction

      GridFunction to modify

    • fn: (coords: number[]) => number

      Function that takes coordinates [x, y?, z?] and returns a scalar value

    Returns void

    Throws

    Error if the grid function has been destroyed

    Throws

    Error if projection fails

    Example: Project sine wave

    projectFunction(gf, ([x, y]) => Math.sin(Math.PI * x) * Math.sin(Math.PI * y));

    Example: Project manufactured solution

    projectFunction(solution, ([x, y, z]) => {
      return x * x + y * y - 2 * z;
    });

    Example: Project radial function

    projectFunction(gf, ([x, y]) => {
      const r = Math.sqrt(x * x + y * y);
      return Math.exp(-r * r);
    });
