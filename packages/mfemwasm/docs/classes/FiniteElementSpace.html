Class FiniteElementSpace

High-level wrapper for MFEM FiniteElementSpace objects.

A finite element space defines the function space used for discretizing PDEs. It combines a mesh with a choice of finite element basis functions (e.g., H1, L2, H(curl), H(div)) and polynomial order.

Important: Resources must be explicitly freed by calling destroy().

Example: Creating an H1 space for scalar problems

import { Mesh } from 'mfemwasm';
import { FiniteElementSpace } from 'mfemwasm/fespace';

const mesh = await Mesh.fromString(meshData);

// Create a linear (order 1) H1 space
const fespace = await FiniteElementSpace.createH1(mesh, 1);
console.log(`Degrees of freedom: ${fespace.ndofs}`);

// Clean up
fespace.destroy();
mesh.destroy();

Example: Creating a vector-valued H1 space

// For elasticity: 2D mesh with 2-component displacement field
const mesh = await makeCartesian2D(10, 10);
const fespace = await FiniteElementSpace.createH1(mesh, 2, 2); // order=2, vdim=2

console.log(`Vector dimension: ${fespace.vdim}`);
console.log(`Total DOFs: ${fespace.ndofs}`);

destroy

  • destroy(): void

    Destroys this finite element space and frees all associated WASM memory.

    After calling this method, the space object should not be used. Any subsequent method calls will throw an error.

    Important: Always call this method when you are done with a space to prevent memory leaks. The associated mesh should be destroyed separately.

    Returns void

    Example

    const mesh = await Mesh.fromString(meshData);
    const fespace = await FiniteElementSpace.createH1(mesh, 1);
    try {
      // ... use fespace
    } finally {
      fespace.destroy();
      mesh.destroy();
    }

StaticcreateH1

  • createH1(mesh: Mesh, order: number, vdim?: number): Promise<FiniteElementSpace>

    Creates an H1 (continuous Lagrange) finite element space.

    H1 spaces use continuous piecewise polynomial functions. They are the most common choice for scalar problems like heat transfer, and for vector problems like elasticity.

    Basis functions:

    • Order 1: Linear (nodal values at vertices)
    • Order 2: Quadratic (nodal values at vertices and edge midpoints)
    • Order p: Polynomial of degree p

    Parameters

    • mesh: Mesh

      The mesh on which to define the space

    • order: number

      Polynomial order (1 = linear, 2 = quadratic, etc.)

    • vdim: number = 1

      Vector dimension (default 1 for scalar fields, 2 or 3 for vector fields)

    Returns Promise<FiniteElementSpace>

    A promise that resolves to the new FiniteElementSpace

    Throws

    Error if space creation fails

    Example: Scalar Poisson problem

    const mesh = await makeCartesian2D(10, 10);
    const fespace = await FiniteElementSpace.createH1(mesh, 2); // Quadratic H1

    Example: 2D Elasticity (vector-valued)

    const mesh = await makeCartesian2D(10, 10);
    const fespace = await FiniteElementSpace.createH1(mesh, 1, 2); // Linear, 2 components
